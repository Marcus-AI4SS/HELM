Install

Build HELM from source.

Public binary artifacts are prepared through the release checklist. Until a tagged release is available, use the source workflow below. 

cd apps/desktop
npm install
npm run build
cd src-tauri
cargo check

Run release checks

powershell -ExecutionPolicy Bypass -File release/check-public-release.ps1

First launch

HELM can start without a live environment. If no trusted projects are detected, it shows onboarding actions: copy a Codex handoff, open Codex, or inspect environment status.
