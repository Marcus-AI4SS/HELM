零基础教程

打开 HELM 后，知道该点哪里。

HELM 是本地项目看板。它不聊天、不写作、不创建项目，也不会替你自动研究。它帮你看本机项目状态，并把下一步要交给 Codex 的说明整理出来。

不知道点哪里时

先看你想完成什么，再去对应页面。

我还没有项目

去“项目”页，点击“复制接入说明”，再把说明和真实项目路径交给 Codex。

我想换项目

使用顶部“当前项目”，或在“项目”页选择一个可读取项目。

我想看材料够不够

去“证据”页，查看材料和依据是否已经读到。

我想继续交给 Codex

去“交给 Codex”页，复制整理好的说明，再回 Codex 发送。

我想打开文件

去“文件”页，打开已有文稿、图表、报告或整理包。

我怀疑电脑有问题

去“本机”页，运行本地检查，或复制本机诊断给 Codex。

第一次使用的六步

1. 打开“项目”

先看 HELM 是否已经看到你的项目。如果已经看到，直接看状态。

2. 复制接入说明

如果没有项目，复制接入说明。这只是复制文字，不会改动文件。

3. 回到 Codex

把说明粘贴给 Codex，并补上真实项目文件夹位置。

4. 回 HELM 刷新

Codex 准备好本地项目上下文后，回到 HELM 点击刷新。

5. 看五个页面

项目、证据、交给 Codex、文件、本机分别回答不同问题。

6. 继续回 Codex

需要下一步时，在“交给 Codex”页复制说明，再回 Codex 继续。
