Privacy

HELM reads local state and keeps the public payload clean.

This repository has no telemetry service, analytics SDK, hosted sync, or cloud backend. HELM renders local project state on your machine and keeps public release artifacts separate from personal data.

Allowed locally

Project maps, evidence ledgers, material indexes, deliverable indexes, validator reports, and local settings.

Excluded from public release

Personal outputs, note vaults, browser profiles, reference databases, credentials, keys, cookies, and local paths.

Release scripts include a privacy scan before artifacts are prepared.
