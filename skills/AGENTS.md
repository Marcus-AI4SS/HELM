# HELM Public Repository Rules

This repository is the public source boundary for HELM.

## Constraints

- Do not add personal project materials.
- Do not add credentials, tokens, cookies, browser profiles, or local service secrets.
- Do not add reference-manager databases or note vaults.
- Do not add local absolute paths.
- Keep HELM as a local dashboard and handoff surface. Do not turn it into a chat app, research executor, agent scheduler, manuscript writer, citation verifier, or submission tool.

## Truth Sources

HELM can display project truth sources when a local project provides them:

- `research-map.md`
- `material-passport.yaml`
- `evidence-ledger.yaml`
- `findings-memory.md`

The public repository may include schemas and sample-safe resources only.
