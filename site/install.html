安装

先从源码运行 HELM。

HELM 当前处于公开候选状态。你可以先从源码运行；正式安装包会在公开发布时附带校验信息。

Windows

需要 Node.js、Rust 和 Windows WebView2。公开安装包发布前，请使用下面的 PowerShell 命令本地运行。

macOS

需要 Node.js、Rust 和 Tauri 所需系统组件。使用同一个 HELM 源码仓库，不需要维护者环境。

打开以后

如果 HELM 没有项目，复制“项目”页面的接入说明，交给 Codex 接入项目。

Windows PowerShell

1 安装依赖 
cd apps/desktop
npm install
2 构建、检查并启动 
npm run build
cd src-tauri
cargo check
cd ..
npm run tauri -- dev
3 运行公开检查 
cd ..\..
powershell -ExecutionPolicy Bypass -File release/check-public-release.ps1

macOS shell

1 安装依赖 
cd apps/desktop
npm install
2 构建、检查并启动 
npm run build
cd src-tauri
cargo check
cd ..
npm run tauri -- dev
3 公开检查

公开检查脚本是 PowerShell 脚本。macOS 可在安装 `pwsh` 后从仓库根目录运行，或使用 GitHub Actions 的公开检查结果。

cd ../..
pwsh -File release/check-public-release.ps1

构建本地安装包

构建脚本会记录签名状态和 SHA-256 校验和；未签名包会明确保持未签名状态。

powershell -ExecutionPolicy Bypass -File release/build-public-artifacts.ps1
