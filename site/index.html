First Run

Open HELM, then let Codex connect your project.

HELM does not ask you to type prompts or create a project inside the app. It reads local project context after Codex has prepared the facts in your project folder.

  1. Install HELMBuild from source today, or use a public candidate artifact when available.
  2. Copy the intake templateIf no project appears, use the Project page button.
  3. Paste it into CodexCodex connects the folder; HELM shows the local status after refresh.

Public Boundary

A dashboard, not a research engine.

  • No hosted telemetry or cloud sync in this repository.
  • No prompt input, chat surface, or hidden agent scheduling.
  • No bundled personal projects, note vaults, browser state, or reference databases.
  • Runs without VELA by showing an empty or public-safe sample state.
HELM is the command dashboard. VELA provides compatible local workflow context.
