Privacy

HELM is local-first.

This repository does not include telemetry, analytics, hosted sync, or a cloud backend. HELM reads local state and renders it on your machine.

What should not be in a public release

  • Personal research outputs.
  • Browser profiles or session state.
  • Reference-manager databases.
  • Note vaults or unpublished project files.
  • Credentials, keys, cookies, or local service secrets.

See the repository privacy policy for the exact release boundary.
