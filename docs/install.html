Install

Build from source today. Signed releases come later.

HELM is a public candidate. Source build checks are available now; signed Windows and macOS artifacts are tracked in the release checklist.

1. Dependencies

cd apps/desktop
npm install

2. Frontend

npm run build

3. Tauri

cd src-tauri
cargo check

Release Gate

The public gate checks privacy boundaries and release readiness.

powershell -ExecutionPolicy Bypass -File release/check-public-release.ps1
