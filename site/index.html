Boundary

Local-first, public-safe, deliberately narrow.

  • No telemetry service in this repository.
  • No chat box, prompt field, or hidden agent scheduler.
  • No bundled personal projects, note vaults, browser state, or reference databases.
  • Works without VELA by showing an empty or sample state.
HELM is a local command dashboard. VELA provides compatible workflow context when available.
